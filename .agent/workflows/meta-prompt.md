# Meta Prompt

Improve a prompt using best practices for structure and clarity.

## Steps

* Analyze the original prompt provided
* Apply prompt engineering best practices:
  - Define a clear **Role** for the AI
  - State the **Objective** precisely
  - Specify **Constraints** and rules
  - Define the expected **Output Format**
  - Include **Examples** if helpful
* Keep the scope exactly as requested — do not add or remove objectives
* Return the improved prompt in markdown format, ready to use
